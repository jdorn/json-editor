# JSON Editor

JSON Schema -> HTML Editor -> JSON

## Getting Started
Download the [production version][min] or the [development version][max].

[min]: https://raw.github.com/jdorn/json-editor/master/dist/jsoneditor.min.js
[max]: https://raw.github.com/jdorn/json-editor/master/dist/jsoneditor.js

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/jsoneditor.min.js"></script>
<script>
jQuery(function($) {
  $.awesome(); // "awesome"
});
</script>
```
